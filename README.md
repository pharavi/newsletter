# Resume Builder

Enter a job description or a job title and get suggestions like career objective, skills and some Achievements to put in your resume.

TechStack:

- NextJS
- TailwindCSS
- GPT-3

How to start
1- Open repo in vscode
2- Open new terminal
3- npm i
4- npm run dev

Application will be live on  http://localhost:3000
